Ticket 3.1: The ImGui 3D Projection Utility
Goal: Implement the math to convert rhythm game time into 3D depth, and then project that into 2D screen space.

Step 1: Define Viewport Constants
Inside the GuitarSong class, define the constants that control the camera's perspective:

C#
private const float VanishingPointX = 640f; // Center of the screen
private const float HorizonY = 200f;        // Where notes spawn (top)
private const float HitLineY = 650f;        // Where notes are played (bottom)
private const float FocalLength = 1000f;    // Controls the "zoom" or distortion
private const float StringSpacing = 40f;    // Distance between strings at the HitLine
private const float ScrollSpeed = 0.5f;     // Z-units per millisecond
Step 2: The Projection Method
Implement the following utility method. The logic is: Calculate how far away the note is in Z-space, determine the perspective scale factor, and map the X/Y coordinates to the screen.

C#
private System.Numerics.Vector2 ProjectToScreen(int stringIndex, float noteTimeMs, float currentAudioTimeMs) {
    // 1. Calculate Depth (Z)
    // Notes in the future have a positive Z. Notes exactly at the hit line have Z = 0.
    float timeDiff = noteTimeMs - currentAudioTimeMs;
    float zDepth = timeDiff * ScrollSpeed;
    
    // Prevent division by zero or negative rendering behind the camera
    if (zDepth < -FocalLength) return new Vector2(-1, -1); 

    // 2. Calculate Perspective Scale
    // Scale is 1.0 at the hit line (Z=0). As Z increases, Scale approaches 0.
    float scale = FocalLength / (FocalLength + zDepth);

    // 3. Calculate World X (Centered around 0)
    // Map string indices 0-5 to X coordinates: -2.5, -1.5, -0.5, 0.5, 1.5, 2.5
    float centerOffset = (stringIndex - 2.5f) * StringSpacing;

    // 4. Project to 2D Screen
    float screenX = VanishingPointX + (centerOffset * scale);
    
    // Y moves linearly from HitLine to Horizon based on scale
    // At scale 1.0 (HitLine), Y = HitLineY. At scale 0.0 (Infinity), Y = HorizonY
    float screenY = HorizonY + ((HitLineY - HorizonY) * scale);

    return new System.Numerics.Vector2(screenX, screenY);
}
